去而复返

发表于 2019-05-17
字数统计: 958 | 阅读时长 ≈ 3 min
好了，历经了约半个月的锤打敲炼，这个博客至此算是搭建并优化完成了。

  (当然，我是建议一边听音乐一边看这篇文章的)

  并没有想象中的哪么开心，反而觉得：“我的个人博客就应该是这个样子”，它在我脑海里已经有了样式、形态，近半个月做的事情不过是把这些想法一点点实现了，其中离不开许多文章的帮助，如果要归纳一下这些文章的来源、出处的话，大致来自这些地方：知乎、简书、CSDN、类似博客上的总结文章、GitHub等等，当然不能少了谷歌搜索，不然我没法准确地找到这些参考资料。

  如果要思考搭建这个博客的初衷的话，可能是想在一个特别的地方输出一些文字等等，然后在推特上偶然看到一位朋友自建的博客，于是乎我就迫不及待地行动起来了，他好像是借助了Wordpress，我也忘记了是怎么找到另一个方法（托管到GitHub），确定大方向后，就在网络上四处扒取文章资料来实现这个构想。

  现在在此写下这个博客第一篇还算正式的文章，觉得自己在做一件似可有可无，又好像很正式的事情一样。想了想，有些时候过短的文字无法承担起一些思考，而在博客上自己可以这么随意洋洋洒洒写下去，也可以逻辑严密地编排一些思考。在这个博客里，我决定写些这些方面的内容：一些读书笔记，分享些音乐（迫于上传限制，大多数是mp3格式），写写影评或者是一些动漫作品的观后感，用来记录日常生活就当写日记，记录下自己学习某些东西的过程、心得、体会等等。

  总的来说就是写些随笔，分享所见所思，如果时间允许、自己创作欲旺盛的话，我觉得也可以在这个博客上写下自己构思的一些小故事之类，为生活添些乐子。

  目前这个时候，这个博客对外界是不可见的，通过搜索引擎怕是很难找到这个小地方，之前在推特上还发布了博客的链接，不过刚刚我把它移出了，应该不会有他人发现这个或推测出这个地址，在当下，我只想把这个博客作为一个个人消遣的地方，如果真的有人发现了的话，em… 无所谓了，这也不同于空间、微博或是别的地方，今天的主页和昨天的并不会有多大不同、这周与上周、这月与上月… 如果我懒的话，这个博客可能会闲置很长时间，不过我会尽力不让这件事情发生的，所以呢，给自己立个目标：每周至少写一篇文章。

  咳咳，我怎么觉得自己一开始码字，整个文风就严肃起来了（可能是这工科读的），当然也没什么地方好活泼着来写的，希望自己哪怕弃了这个博客的大坑，也要一直保持思考，持续学习，如果不能坚持这一点，也就不会有这个博客的成功建成了，那么，就写到这儿吧，希望自己能多多耕耘⚚

  (某些情况下，侧栏的音乐可能无法正常播放，所以我把它贴在了这篇博文的开头那边)
